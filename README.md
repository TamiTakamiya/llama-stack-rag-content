# llama-stack-rag-content

## Setup

```commandline
git clone git@github.com:TamiTakamiya/llama-stack-rag-content.git
uv init -p 3.11 llama-stack-rag-content
python3.11 -m venv .venv
uv pip install llama-stack

```
